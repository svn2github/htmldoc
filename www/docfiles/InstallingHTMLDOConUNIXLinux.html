Installing HTMLDOC on UNIX/Linux

Run the following commands as the root user to install HTMLDOC on UNIX/Linux:

    gunzip htmldoc-version-platform.tar.gz ENTER
    tar xf htmldoc-version-platform.tar ENTER
    ./setup ENTER

Substitute the correct version and platform strings as appropriate. After you have run the setup program, the window shown in Figure 1-5 will appear. Follow the installation prompts until the software is installed.
