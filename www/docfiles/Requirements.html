Requirements

HTMLDOC requires C and C++ compilers - recent versions of GCC/LLVM work fine. To build the GUI you'll also need:

  • Fast Light Tool Kit ("FLTK"), version 1.1.x or 1.3.x.
  • X11 libraries, R5 or higher (needed to build under UNIX/Linux only.)

Secure (https) URL support can be enabled via the OpenSSL or GNU TLS libraries, among others.
